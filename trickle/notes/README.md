# Project README

## Overview
This is a portfolio website for a Senior AI/LLM Engineer, optimized for Japanese IT startups (Small to Mid-size).
The content focuses on "Production Readiness", "Security", and "Trustworthiness".

## Key Sections
- **Hero**: Strengthened first line for immediate impact (Business Value + Production Focus).
- **Summary**: 1-page summary for CEOs.
- **Skills**: 3-level breakdown (Expert, Proficient, Familiar).
- **Projects**: Fact-based case studies with unified format and clear "Responsibility Scope".
- **Videos**: Guided viewing experience for demo videos.
- **Work Style**: 5 key values for remote/Japanese team collaboration (Reporting, Goal Setting, Risk Mgmt).

## Maintenance
- Keep the language simple and concise (20-40 chars/line).
